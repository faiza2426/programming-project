# Bank Management System (C++)

This is a console-based Bank Management System developed in C++ using Visual Studio.

## Features
- Admin login
- User account request
- Account approval by admin
- Deposit money
- Withdraw money
- Transfer money between accounts
- Check balance
- Transaction history
- File handling for data storage
